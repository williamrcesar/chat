# Notification sounds

Sons padrão (opcionais). Se não existirem, o app usa um beep curto (Web Audio).

- `notification.mp3` — padrão
- `ding.mp3` — sino
- `chime.mp3` — chime
- `pop.mp3` — pop
- `soft.mp3` — suave

Cada conversa pode usar um desses ou um áudio customizado (máx. 6 segundos) nas configurações da conversa.
